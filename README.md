# terraform-basics
terraform basics

## dir structure
```
ec2-instance/
├── main.tf
├── variables.tf
├── terraform.tfvars
├── outputs.tf 
```

## commands

`tfenv install 1.12.2`

`tfenv use 1.12.2`

`terraform --version`

`terraform init`

`terraform plan -out tfplan`

### to clean up everything

`terraform deploy`
